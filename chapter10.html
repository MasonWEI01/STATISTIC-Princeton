Chapter 10: Tools: Convolution and Variable Substitution (工具：卷积和变量替换)

Definitions (定义)

Definition: Convolution

The convolution of two functions f and g, denoted by (f * g), is defined as:

(f * g)(t) = ∫-∞∞ f(τ)g(t-τ) dτ

两个函数f和g的卷积，记为(f * g)，定义为上述积分。

Definition: Sum of Independent Random Variables

If X and Y are independent random variables with probability density functions fX and fY, then the probability density function of Z = X + Y is the convolution of fX and fY:

fZ(z) = (fX * fY)(z) = ∫-∞∞ fX(x)fY(z-x) dx

如果X和Y是具有概率密度函数fX和fY的独立随机变量，则Z = X + Y的概率密度函数是fX和fY的卷积。

Definition: Variable Substitution

If X is a random variable with probability density function fX, and Y = g(X) is a function of X, then the probability density function of Y can be found using the change of variable formula.

如果X是具有概率密度函数fX的随机变量，Y = g(X)是X的函数，则Y的概率密度函数可以使用变量替换公式求得。

Theorems (定理)

Theorem: Convolution of PMFs

If X and Y are independent discrete random variables with PMFs pX and pY, then the PMF of Z = X + Y is:

pZ(z) = ∑x pX(x)pY(z-x)

Where the sum is over all possible values of X.

Theorem: Convolution of PDFs

If X and Y are independent continuous random variables with PDFs fX and fY, then the PDF of Z = X + Y is:

fZ(z) = ∫-∞∞ fX(x)fY(z-x) dx

Theorem: Change of Variable Formula (Univariate)

If X has PDF fX and Y = g(X) where g is a strictly monotonic differentiable function, then the PDF of Y is:

fY(y) = fX(g-1(y)) · |d/dy g-1(y)|

Where g-1 is the inverse function of g.

Theorem: Convolution and MGFs

If X and Y are independent random variables with moment generating functions MX and MY, then the moment generating function of Z = X + Y is:

MZ(t) = MX(t) · MY(t)

Formulas (公式)

Formula: Sum of Independent Normal Random Variables
If X ~ N(μX, σX2) and Y ~ N(μY, σY2) are independent, then:
X + Y ~ N(μX + μY, σX2 + σY2)
Formula: Linear Transformation of a Random Variable
If X has PDF fX and Y = aX + b (a ≠ 0), then:
fY(y) = (1/|a|) · fX((y-b)/a)
Formula: Sum of Independent Poisson Random Variables
If X ~ Poisson(λ1) and Y ~ Poisson(λ2) are independent, then:
X + Y ~ Poisson(λ1 + λ2)
Formula: Convolution of Exponential Random Variables
If X ~ Exp(λ) and Y ~ Exp(λ) are independent, then:
X + Y ~ Gamma(2, λ)

More generally, the sum of n independent Exp(λ) random variables follows a Gamma(n, λ) distribution.
