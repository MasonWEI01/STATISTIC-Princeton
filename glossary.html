Glossary (术语表)

This glossary provides Chinese-English translations of specialized terminology in probability theory, focusing on terms from the Princeton Guide to Probability Theory.

Part 2: Random Variables

Random Variable
随机变量
A function that assigns a real number to each outcome in the sample space of a random experiment.
Probability Mass Function (PMF)
概率质量函数
A function that gives the probability that a discrete random variable is exactly equal to some value.
Probability Density Function (PDF)
概率密度函数
A function whose value at any given sample point in the sample space can be interpreted as providing a relative likelihood that the value of the random variable would equal that sample.
Cumulative Distribution Function (CDF)
累积分布函数
A function that gives the probability that a random variable X takes on a value less than or equal to x.
Expected Value
期望值
The weighted average of all possible values that a random variable can take on, where the weights are the probabilities of taking on those values.
Variance
方差
A measure of how far a set of numbers is spread out from their average value.
Standard Deviation
标准差
A measure of the amount of variation or dispersion of a set of values, defined as the square root of the variance.
Convolution
卷积
A mathematical operation on two functions that produces a third function expressing how the shape of one is modified by the other.
Moment Generating Function
矩母函数
A function that generates the moments of a probability distribution.

Part 3: Advanced Topics

Markov Chain
马尔可夫链
A stochastic model describing a sequence of possible events in which the probability of each event depends only on the state attained in the previous event.
Random Walk
随机游走
A mathematical object that describes a path that consists of a succession of random steps on some mathematical space.
Bivariate Random Variable
二维随机变量
A pair of random variables that maps each element in the sample space to a point in the two-dimensional real plane.
Gamma Function
伽马函数
An extension of the factorial function to real and complex numbers.
Chi-Square Distribution
卡方分布
A distribution of a sum of the squares of k independent standard normal random variables.
t-Distribution
t分布
A continuous probability distribution that arises when estimating the mean of a normally distributed population in situations where the sample size is small and the population standard deviation is unknown.
Central Limit Theorem
中心极限定理
A theorem that states that, under certain conditions, the sum of a large number of random variables will have an approximately normal distribution.
Law of Large Numbers
大数定律
A theorem that describes the result of performing the same experiment a large number of times, stating that the average of the results will be close to the expected value.
Statistical Estimation
统计估计
The process of using sample data to estimate the parameters of a population.
Hypothesis Testing
假设检验
A statistical method that is used in making statistical decisions using experimental data.
