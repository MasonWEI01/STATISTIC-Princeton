Glossary (术语表)

This glossary provides Chinese-English translations of specialized terminology in probability theory, focusing on terms from Part 2 of the Princeton Guide to Probability Theory.

Random Variable
随机变量
A function that assigns a real number to each outcome in the sample space of a random experiment.
Probability Mass Function (PMF)
概率质量函数
A function that gives the probability that a discrete random variable is exactly equal to some value.
Probability Density Function (PDF)
概率密度函数
A function whose value at any given sample point in the sample space can be interpreted as providing a relative likelihood that the value of the random variable would equal that sample.
Cumulative Distribution Function (CDF)
累积分布函数
A function that gives the probability that a random variable X takes on a value less than or equal to x.
Expected Value
期望值
The weighted average of all possible values that a random variable can take on, where the weights are the probabilities of taking on those values.
Variance
方差
A measure of how far a set of numbers is spread out from their average value.
Standard Deviation
标准差
A measure of the amount of variation or dispersion of a set of values, defined as the square root of the variance.
Convolution
卷积
A mathematical operation on two functions that produces a third function expressing how the shape of one is modified by the other.
Moment Generating Function
矩母函数
A function that generates the moments of a probability distribution.
Law of Large Numbers
大数定律
A theorem that describes the result of performing the same experiment a large number of times, stating that the average of the results will be close to the expected value.
