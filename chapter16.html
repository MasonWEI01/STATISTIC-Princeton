Chapter 16: Chi-Square Distribution (卡方分布)

Definitions (定义)

Definition: Chi-Square Distribution

If Z₁, Z₂, ..., Zₙ are n independent standard normal random variables, then the random variable X = Z₁² + Z₂² + ... + Zₙ² follows a chi-square distribution with n degrees of freedom, denoted as X ~ χ²(n).

如果Z₁, Z₂, ..., Zₙ是n个独立的标准正态随机变量，则随机变量X = Z₁² + Z₂² + ... + Zₙ²服从自由度为n的卡方分布，记为X ~ χ²(n)。

Definition: Non-central Chi-Square Distribution

If Z₁, Z₂, ..., Zₙ are n independent normal random variables with E[Zᵢ] = μᵢ and Var(Zᵢ) = 1, then the random variable X = Z₁² + Z₂² + ... + Zₙ² follows a non-central chi-square distribution with n degrees of freedom and non-centrality parameter λ = μ₁² + μ₂² + ... + μₙ², denoted as X ~ χ²(n,λ).

如果Z₁, Z₂, ..., Zₙ是n个独立的正态随机变量，其中E[Zᵢ] = μᵢ，Var(Zᵢ) = 1，则随机变量X = Z₁² + Z₂² + ... + Zₙ²服从非中心参数λ = μ₁² + μ₂² + ... + μₙ²的非中心卡方分布，记为X ~ χ²(n,λ)。

Definition: Chi-Square Test

The chi-square test is a statistical hypothesis test used to determine whether there is a significant difference between observed frequencies and expected frequencies.

卡方检验是一种统计假设检验方法，用于确定观察到的频率与理论频率之间的差异是否显著。

Theorems (定理)

Theorem: Additivity of Chi-Square Distribution

If X ~ χ²(n) and Y ~ χ²(m) are independent random variables, then X + Y ~ χ²(n+m).

Theorem: Relation Between Chi-Square and Gamma Distributions

A chi-square distribution with n degrees of freedom is equivalent to a gamma distribution with shape parameter α = n/2 and scale parameter β = 2.

Theorem: Distribution of Square of Standard Normal

If Z ~ N(0,1), then Z² ~ χ²(1).

Theorem: Distribution of Sample Variance

If X₁, X₂, ..., Xₙ is an independent and identically distributed random sample from a normal distribution N(μ,σ²), then (n-1)S²/σ² ~ χ²(n-1), where S² is the sample variance.

Formulas (公式)

Formula: Probability Density Function of Chi-Square Distribution
The probability density function of a chi-square distribution with n degrees of freedom is:
f(x) = (1/(2^(n/2)Γ(n/2)))x^(n/2-1)e^(-x/2), where x > 0
Formula: Expected Value and Variance of Chi-Square Distribution
If X ~ χ²(n), then:
E[X] = n
Var(X) = 2n
Formula: Moment Generating Function of Chi-Square Distribution
If X ~ χ²(n), then its moment generating function is:
M_X(t) = (1-2t)^(-n/2), where t < 1/2
Formula: Probability Density Function of Non-central Chi-Square Distribution
The probability density function of a non-central chi-square distribution with n degrees of freedom and non-centrality parameter λ is:
f(x) = e^(-λ/2)∑_{j=0}^∞ (λ/2)^j/j! · (1/(2^((n+2j)/2)Γ((n+2j)/2)))x^((n+2j)/2-1)e^(-x/2), where x > 0
