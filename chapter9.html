Chapter 9: Tools: Expectation (工具：期望)

Definitions (定义)

Definition: Expected Value

The expected value (or mean) of a random variable X, denoted by E[X], is:

For discrete random variables: E[X] = ∑x x · pX(x)

For continuous random variables: E[X] = ∫-∞∞ x · fX(x) dx

随机变量X的期望值（或均值），记为E[X]，是X的所有可能值的加权平均，其中权重是相应的概率。

Definition: Variance

The variance of a random variable X, denoted by Var(X) or σX2, is:

Var(X) = E[(X - E[X])²] = E[X²] - (E[X])²

随机变量X的方差，记为Var(X)或σX2，是X与其期望值之差的平方的期望值。

Definition: Standard Deviation

The standard deviation of a random variable X, denoted by σX, is the square root of the variance:

σX = √Var(X)

随机变量X的标准差，记为σX，是方差的平方根。

Definition: Moment

The n-th moment of a random variable X is defined as:

E[Xn]

随机变量X的n阶矩定义为X的n次方的期望值。

Definition: Moment Generating Function

The moment generating function (MGF) of a random variable X is defined as:

MX(t) = E[etX]

随机变量X的矩母函数定义为etX的期望值。

Theorems (定理)

Theorem: Linearity of Expectation

For any random variables X and Y, and any constants a and b:

E[aX + bY] = aE[X] + bE[X]

Theorem: Variance of Linear Combinations

For independent random variables X and Y, and any constants a and b:

Var(aX + bY) = a²Var(X) + b²Var(Y)

Theorem: Covariance

The covariance of two random variables X and Y is defined as:

Cov(X,Y) = E[(X - E[X])(Y - E[Y])] = E[XY] - E[X]E[Y]

Theorem: Properties of MGF

If MX(t) is the moment generating function of X, then:

  1. MX(0) = 1
  2. The n-th derivative of MX(t) evaluated at t=0 gives the n-th moment of X:

    MX(n)(0) = E[Xn]

Formulas (公式)

Formula: Expected Value of a Function of a Random Variable
E[g(X)] = ∑x g(x) · pX(x) (discrete case)
E[g(X)] = ∫-∞∞ g(x) · fX(x) dx (continuous case)
Formula: Chebyshev's Inequality
P(|X - E[X]| ≥ kσX) ≤ 1/k²

Where k > 0 and σX is the standard deviation of X.

Formula: Correlation Coefficient
ρX,Y = Cov(X,Y)/(σXσY)

Where σX and σY are the standard deviations of X and Y, respectively.

Formula: Law of Total Expectation
E[X] = E[E[X|Y]]

Where E[X|Y] is the conditional expectation of X given Y.
