Chapter 7: Discrete Random Variables (离散型随机变量)

Definitions (定义)

Definition: Random Variable

A random variable is a function that assigns a real number to each outcome in the sample space of a random experiment.

随机变量是一个函数，它将随机试验的样本空间中的每个结果分配一个实数。

Definition: Probability Mass Function (PMF)

For a discrete random variable X, the probability mass function pX(x) gives the probability that X takes on the value x:

pX(x) = P(X = x)

对于离散型随机变量X，概率质量函数pX(x)给出X取值为x的概率。

Definition: Cumulative Distribution Function (CDF)

The cumulative distribution function FX(x) of a random variable X is defined as:

FX(x) = P(X ≤ x)

随机变量X的累积分布函数FX(x)定义为X不大于x的概率。

Theorems (定理)

Theorem: Properties of PMF

For any discrete random variable X with probability mass function pX(x):

  1. pX(x) ≥ 0 for all x
  2. ∑x pX(x) = 1, where the sum is over all possible values of X
Theorem: Relationship between PMF and CDF

For a discrete random variable X:

FX(x) = ∑t≤x pX(t)

Where the sum is over all possible values t of X such that t ≤ x.

Formulas (公式)

Formula: Expected Value of a Discrete Random Variable
E[X] = ∑x x · pX(x)

Where the sum is over all possible values x of X.

Formula: Variance of a Discrete Random Variable
Var(X) = E[(X - E[X])²] = E[X²] - (E[X])²
Formula: Bernoulli Distribution PMF
pX(x) = px(1-p)1-x for x ∈ {0,1}

Where p is the probability of success.

Formula: Binomial Distribution PMF
pX(x) = (n choose x) px(1-p)n-x for x ∈ {0,1,2,...,n}

Where n is the number of trials and p is the probability of success in each trial.

Formula: Geometric Distribution PMF
pX(x) = (1-p)x-1p for x ∈ {1,2,3,...}

Where p is the probability of success in each trial.

Formula: Poisson Distribution PMF
pX(x) = e-λλx/x! for x ∈ {0,1,2,...}

Where λ is the average rate of occurrence.
