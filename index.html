Welcome to the interactive website for the Princeton Guide to Probability Theory. This website presents the core content of the book in an organized, accessible format, focusing on definitions, formulas, and theorems.

About This Website

This website is designed to help you understand and quickly absorb the essence of probability theory. It focuses on Part 2 of the book, which covers statistical knowledge through random variables and related concepts.

How to Use This Website

Use the navigation menu on the left to browse through different chapters. Each chapter contains key definitions, theorems, lemmas, and formulas from the original text, presented in a clear, structured format.

The glossary provides Chinese-English translations of specialized terminology to aid in understanding the concepts.

Featured Content

Definition (Random Variable)

A random variable is a function that assigns a real number to each outcome in the sample space of a random experiment.

Theorem (Law of Large Numbers)

As the number of trials increases, the average of the results will converge to the expected value.
