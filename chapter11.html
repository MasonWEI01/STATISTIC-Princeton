Chapter 11: Tools: Differential Identities (工具：微分恒等式)

Definitions (定义)

Definition: Differential Identity

A differential identity is an equation that relates a function to its derivatives.

微分恒等式是将函数与其导数相关联的等式。

Definition: Characteristic Function

The characteristic function of a random variable X is defined as:

φX(t) = E[eitX] = ∫-∞∞ eitxfX(x) dx

Where i is the imaginary unit and fX is the PDF of X.

随机变量X的特征函数定义为上述期望值或积分。

Definition: Cumulant Generating Function

The cumulant generating function of a random variable X is defined as:

KX(t) = log(MX(t))

Where MX(t) is the moment generating function of X.

随机变量X的累积量生成函数定义为其矩母函数的自然对数。

Theorems (定理)

Theorem: Derivatives of Characteristic Function

If φX(t) is the characteristic function of X, then:

  1. φX(0) = 1
  2. The n-th derivative of φX(t) evaluated at t=0 gives the n-th moment of X multiplied by in:

    φX(n)(0) = inE[Xn]

Theorem: Inversion Formula

If φX(t) is the characteristic function of a continuous random variable X with PDF fX, then:

fX(x) = (1/2π) ∫-∞∞ e-itxφX(t) dt

Theorem: Derivatives of Cumulant Generating Function

If KX(t) is the cumulant generating function of X, then:

  1. KX(0) = 0
  2. KX'(0) = E[X]
  3. KX''(0) = Var(X)
Theorem: Characteristic Function of Sum of Independent Random Variables

If X and Y are independent random variables with characteristic functions φX and φY, then the characteristic function of Z = X + Y is:

φZ(t) = φX(t) · φY(t)

Formulas (公式)

Formula: Characteristic Function of Normal Distribution
If X ~ N(μ, σ²), then:
φX(t) = exp(iμt - σ²t²/2)
Formula: Characteristic Function of Exponential Distribution
If X ~ Exp(λ), then:
φX(t) = λ/(λ - it)
Formula: Characteristic Function of Poisson Distribution
If X ~ Poisson(λ), then:
φX(t) = exp(λ(eit - 1))
Formula: Cumulant Generating Function of Normal Distribution
If X ~ N(μ, σ²), then:
KX(t) = μt + σ²t²/2
