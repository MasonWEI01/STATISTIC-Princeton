Chapter 8: Continuous Random Variables (连续型随机变量)

Definitions (定义)

Definition: Continuous Random Variable

A continuous random variable is a random variable that can take on any value in a continuous range (interval) of values.

连续型随机变量是一种可以取连续范围（区间）内任意值的随机变量。

Definition: Probability Density Function (PDF)

For a continuous random variable X, the probability density function fX(x) is a function such that:

P(a ≤ X ≤ b) = ∫ab fX(x) dx

对于连续型随机变量X，概率密度函数fX(x)是一个函数，使得X落在区间[a,b]内的概率等于fX(x)在该区间上的积分。

Definition: Cumulative Distribution Function (CDF)

The cumulative distribution function FX(x) of a continuous random variable X is defined as:

FX(x) = P(X ≤ x) = ∫-∞x fX(t) dt

连续型随机变量X的累积分布函数FX(x)定义为X不大于x的概率，等于概率密度函数在负无穷到x的积分。

Theorems (定理)

Theorem: Properties of PDF

For any continuous random variable X with probability density function fX(x):

  1. fX(x) ≥ 0 for all x
  2. ∫-∞∞ fX(x) dx = 1
Theorem: Relationship between PDF and CDF

For a continuous random variable X:

fX(x) = d/dx FX(x)

That is, the probability density function is the derivative of the cumulative distribution function.

Formulas (公式)

Formula: Expected Value of a Continuous Random Variable
E[X] = ∫-∞∞ x · fX(x) dx
Formula: Variance of a Continuous Random Variable
Var(X) = E[(X - E[X])²] = ∫-∞∞ (x - E[X])² · fX(x) dx
Formula: Uniform Distribution PDF
fX(x) = 1/(b-a) for x ∈ [a,b], 0 otherwise

Where a and b are the minimum and maximum values of the distribution.

Formula: Normal Distribution PDF
fX(x) = (1/√(2πσ²)) · e-(x-μ)²/(2σ²)

Where μ is the mean and σ² is the variance.

Formula: Exponential Distribution PDF
fX(x) = λe-λx for x ≥ 0, 0 otherwise

Where λ > 0 is the rate parameter.
