Chapter 15: Gamma Function (伽马函数)

Definitions (定义)

Definition: Gamma Function

The Gamma function Γ(s) is defined as:

Γ(s) = ∫0∞ xs-1e-x dx, where s > 0

伽马函数Γ(s)定义为上述积分，其中s > 0。

Definition: Beta Function

The Beta function B(a,b) is defined as:

B(a,b) = ∫01 ta-1(1-t)b-1 dt, where a > 0, b > 0

贝塔函数B(a,b)定义为上述积分，其中a > 0，b > 0。

Definition: Incomplete Gamma Function

The incomplete Gamma function γ(s,x) is defined as:

γ(s,x) = ∫0x ts-1e-t dt

不完全伽马函数γ(s,x)定义为上述积分。

Definition: Gamma Distribution

The Gamma distribution is a continuous probability distribution with probability density function:

f(x;α,β) = (βα/Γ(α))xα-1e-βx, where x > 0, α > 0 is the shape parameter, and β > 0 is the rate parameter

伽马分布是一种连续概率分布，其概率密度函数如上所示，其中x > 0，α > 0是形状参数，β > 0是速率参数。

Theorems (定理)

Theorem: Recursive Formula for Gamma Function

For any s > 0:

Γ(s+1) = s·Γ(s)

Theorem: Relation Between Gamma Function and Factorial

For any positive integer n:

Γ(n) = (n-1)!

Theorem: Relation Between Gamma and Beta Functions

For a > 0 and b > 0:

B(a,b) = Γ(a)Γ(b)/Γ(a+b)

Theorem: Stirling's Approximation for Gamma Function

As s approaches infinity:

Γ(s) ≈ √(2π/s)(s/e)s

Theorem: Expected Value and Variance of Gamma Distribution

If a random variable X follows the Gamma distribution Gamma(α,β), then:

E[X] = α/β

Var(X) = α/β²

Formulas (公式)

Formula: Special Value Γ(1/2)
Γ(1/2) = √π
Formula: Duplication Formula for Gamma Function
Γ(2s) = 22s-1Γ(s)Γ(s+1/2)/√π

Γ(s)Γ(1-s) = π/sin(πs), where 0 < s < 1
